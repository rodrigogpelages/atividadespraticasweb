Aula 05: Laboratório Web

Projeto Autoguiado: A Minha Página de Tributo

Prática de Fixação Prof. Rodrigo G. Pelages

Aviso Importante: Proibido Copiar!

Nesta aula, a função "Copiar e Colar" está desativada. O objetivo é que digite o seu próprio código. A memória muscular é a melhor amiga de um programador. Leia atentamente, respire fundo e mãos no teclado!

O Seu Tema: Escolha um Filme, Série, Banda ou Jogo favorito. Ao longo destes 10 passos, substitua os textos genéricos (ex: [Nome do Jogo]) pelas informações reais da sua escolha.
1

Preparando o Ambiente

Antes de programar, precisamos da nossa "mesa de trabalho". Vamos usar um editor de código na nuvem.

  • Abra uma nova aba no seu navegador.
  • Acesse o CodePen (codepen.io/pen) ou JSFiddle (jsfiddle.net).
  • Identifique as janelas: Haverá uma área para HTML e outra para CSS. Os resultados aparecerão automaticamente na tela em branco.
2

A Raiz de Tudo (Estrutura Semântica)

Lembra da Aula 4? O Google precisa entender o que é cabeçalho, conteúdo e rodapé. Vamos criar as caixas invisíveis principais na janela HTML.

Janela HTML
<header> <!-- O cabeçalho vai aqui --> </header> <main> <!-- O conteúdo principal vai aqui --> </main> <footer> <!-- O rodapé vai aqui --> </footer>
3

Títulos Principais (Cabeçalho)

Dentro do seu <header>, vamos adicionar o Título Principal (<h1>) e um pequeno subtítulo/parágrafo (<p>).

Adicione dentro de <header>
<h1>Tributo a: [Seu Tema Aqui]</h1> <p>Uma página criada por [Seu Nome]</p>
4

Escrevendo Artigos (Conteúdo)

Agora, vá para dentro da tag <main>. Vamos usar a tag <article> para criar um bloco de texto com um Título de Nível 2 (<h2>).

Adicione dentro de <main>
<article> <h2>Por que é o meu favorito?</h2> <p>Escreva 2 ou 3 frases sobre a história, personagens ou o motivo de você gostar tanto dessa obra.</p> </article>
5

Listas Não Ordenadas (ul/li)

Logo abaixo do fechamento do seu </article> (mas ainda dentro do <main>), crie uma lista com 3 curiosidades sobre o seu tema.

Abaixo do </article>
<h3>Curiosidades:</h3> <ul> <li>Curiosidade 1...</li> <li>Curiosidade 2...</li> <li>Curiosidade 3...</li> </ul>
6

Adicionando Mídia (Imagens)

Vá ao Google, busque uma imagem do seu tema, clique com o botão direito e selecione "Copiar endereço da imagem". Crie a tag <img> no HTML e não esqueça do alt para leitores de tela!

Abaixo da sua lista </ul>
<img src="COLE-O-LINK-AQUI" alt="Descreva a imagem detalhadamente">
7

Luz, Câmera... (Iframe)

Abra uma nova aba, vá ao YouTube e busque um Trailer do seu tema. Clique em "Compartilhar" > "Incorporar", copie o código e digite-o abaixo da sua imagem.

Abaixo da sua <img>
<iframe width="560" height="315" src="URL-DO-YOUTUBE"></iframe>

Não se esqueça do Rodapé! Vá até a tag <footer> no final do arquivo e adicione um <p>Atividade Aula 5 - 2026</p>.

8

Vestindo a Página (CSS Base)

O seu site deve estar muito feio (imagens gigantes, vídeos tortos). É hora de mudar de janela! Vá para a janela do CSS para aplicar estilos na tag body.

Janela CSS
body { background-color: #e2e8f0; /* Um cinza bem claro */ font-family: Arial, sans-serif; /* Fonte mais limpa */ margin: 0; /* Tira as margens padrão do navegador */ } /* Ajusta as mídias para não quebrarem a tela */ img, iframe { max-width: 100%; border-radius: 10px; }
9

O Box Model na Prática

Lembra da Aula 3? Vamos transformar a tag main numa "caixa flutuante" centralizada, branca e com sombra.

Abaixo do body no CSS
main { background-color: white; max-width: 800px; /* Largura máxima */ margin: 40px auto; /* Auto centraliza a caixa! */ padding: 30px; /* Espaço interno */ border-radius: 15px; /* Cantos arredondados */ box-shadow: 0px 5px 15px rgba(0,0,0, 0.2); }
10

O Toque Final

Para concluir a beleza do seu site, vamos pintar o Cabeçalho e centralizar o Rodapé. Sinta-se à vontade para trocar a cor blueviolet pela cor que mais combine com o seu tema!

Final do arquivo CSS
header { background-color: blueviolet; color: white; text-align: center; padding: 40px; } footer { text-align: center; color: #64748b; padding-bottom: 30px; }

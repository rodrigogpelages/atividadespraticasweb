Aula 05: Laboratório Web

Projeto Autoguiado: A Minha Página de Tributo

Prática de Fixação Prof. Rodrigo G. Pelages

Aviso Importante: Proibido Copiar!

Nesta aula, a função "Copiar e Colar" está desativada. O objetivo é que digite o seu próprio código. A memória muscular é a melhor amiga de um programador. Leia atentamente, respire fundo e mãos no teclado! Os erros fazem parte da aprendizagem.

O Seu Tema: Escolha um Filme, Série, Banda ou Jogo favorito. Ao longo destes 10 passos, substitua os textos genéricos (ex: [Nome do Jogo]) pelas informações reais da sua escolha.
1

Preparando a Mesa de Trabalho

Antes de construirmos uma casa, precisamos do terreno. O nosso terreno será o CodePen, um site onde o código ganha vida instantaneamente.

  • Abra uma nova aba no seu navegador.
  • Acesse o CodePen (codepen.io/pen).
  • Entendendo o ecrã: Verá 3 caixas em cima (HTML, CSS e JS) e um grande espaço branco em baixo.
  • Pode ignorar e minimizar a caixa JS. Hoje só vamos usar a caixa HTML (esqueleto) e a caixa CSS (roupa).
2

As Três Caixas Principais (HTML)

Onde digitar: Na caixa escrita HTML.

Por que estamos a fazer isto? Vamos dividir o nosso site em 3 partes lógicas para o Google entender: o topo (header), o meio (main) e o fundo (footer).

Janela HTML
<header> </header> <main> </main> <footer> </footer>
Atenção ao Fecho! Reparou que a segunda tag tem sempre uma barra /? Isso indica que a caixa fechou. Se esquecer a barra /, o site vai "partir"!
3

O Título da Página

Onde digitar: Exatamente no meio das tags <header> e </header> que acabou de criar.

Por que estamos a fazer isto? A tag h1 (Heading 1) cria o título maior e mais importante do site. A tag p (Paragraph) cria um texto normal.

Dentro de <header>
<h1>Tributo a: [Substitua pelo seu Tema]</h1> <p>Uma página criada por [O Seu Nome]</p>
Olhe para a parte branca em baixo no CodePen. O seu texto já deve ter aparecido! A preto e branco por agora, mas já lá está.
4

Escrevendo o Artigo

Onde digitar: Agora desça e clique no meio das tags <main> e </main>.

Por que estamos a fazer isto? A tag article avisa o navegador que ali começa um texto independente. O h2 é um subtítulo (menor que o h1).

Dentro de <main>
<article> <h2>Por que é o meu favorito?</h2> <p>Gosto muito deste tema porque... (Escreva 2 frases aqui)</p> </article>
5

Criando uma Lista com "Bolinhas"

Onde digitar: Ainda dentro do main, mas depois do </article>.

O que significa? ul significa "Unordered List" (Lista sem números). li significa "List Item" (Item da lista).

Abaixo do </article>
<h3>Curiosidades:</h3> <ul> <li>A primeira curiosidade é...</li> <li>A segunda curiosidade é...</li> <li>A terceira curiosidade é...</li> </ul>
6

Colocando uma Imagem

Como apanhar a imagem: Abra o Google Imagens noutra aba. Pesquise o seu tema. Clique na foto que gostar com o botão direito do rato e escolha "Copiar endereço da imagem".

Onde digitar: Abaixo de onde fechou a lista </ul>.

Abaixo da sua lista </ul>
<img src="COLE-O-LINK-AQUI-DENTRO" alt="Descreva a imagem detalhadamente">
Atenção Tripla!
1. A tag img NÃO tem fecho (não existe </img>).
2. O link da imagem TEM de ficar dentro das aspas " ".
3. Não se esqueça do alt, é obrigatório para pessoas cegas que usam leitores de ecrã!
7

Trazendo um Vídeo do YouTube

Passo a passo no YouTube: Vá ao YouTube e abra um vídeo do seu tema. Em baixo do vídeo, clique em Partilhar > Incorporar (tem um ícone <>). Copie aquele bloco de código enorme que aparece à direita.

Onde colar: Abaixo da sua <img>.

Abaixo da sua <img>
<!-- Cole o código gigantesco do YouTube aqui. Exemplo abaixo: --> <iframe width="560" height="315" src="URL-DO-YOUTUBE" title="YouTube video" frameborder="0"></iframe>
Quase a terminar o HTML! Vá lá abaixo ao final do seu código, encontre a tag <footer> e digite no meio dela: <p>Atividade Aula 5 - 2026</p>.
8

PARE! Vamos Mudar de Janela (CSS)

Se olhar para o resultado agora, o seu site está uma bagunça. As imagens estão gigantes, o fundo é branco e sem graça. A partir de AGORA, vamos digitar APENAS na caixa escrita CSS.

A Regra de Ouro do CSS: Primeiro dizemos quem vamos pintar (ex: body), abrimos chaves {, dizemos o que muda, e terminamos SEMPRE com ponto e vírgula ;.

Na Janela CSS
body { background-color: #e2e8f0; font-family: Arial, sans-serif; margin: 0; } img, iframe { max-width: 100%; border-radius: 10px; }
Cuidado com os Símbolos! Usamos as Chaves { } para envolver as regras. E o ponto e vírgula ; no final de cada linha é como o ponto final numa frase. Se o esquecer, o código "quebra" as linhas seguintes!
9

Criando o Cartão Principal (Main)

Onde digitar: Abaixo do código que acabou de fazer no CSS.

Por que estamos a fazer isto? Vamos transformar a tag main (que engloba o nosso texto e mídias) numa caixa branca flutuante e bonita ao centro do ecrã.

Na Janela CSS (continuando em baixo)
main { background-color: white; max-width: 800px; /* Impede que fique muito esticado */ margin: 40px auto; /* O 'auto' centraliza a caixa no ecrã automaticamente! */ padding: 30px; /* Espaço interno para o texto não colar à borda */ border-radius: 15px; /* Arredonda os cantinhos */ box-shadow: 0px 5px 15px rgba(0,0,0, 0.2); /* Sombra 3D */ }
10

As Cores Finais

Onde digitar: No fim de tudo, ainda na caixa CSS.

Vamos pintar o cabeçalho e centralizar o rodapé. Note que eu usei a cor blueviolet (Roxo Violeta), mas pode apagar e escrever outra cor em inglês (ex: red, darkgreen, black).

No final da Janela CSS
header { background-color: blueviolet; color: white; text-align: center; padding: 40px; } footer { text-align: center; color: #64748b; padding-bottom: 30px; }
